Use "we" instead of "you" when framing questions, suggestions, or calls to action. Say "could we ask the collaborator" not "could you ask".

**Why:** Jeppe views this as joint work; "you" language feels like the assistant is outside the collaboration rather than part of it.

**How to apply:** Any time I am about to write "could you", "you should", "you might want to" — reframe as "we could", "we should", "we might want to".
